### Key conventions

| Convention | Rule |
|-----------|------|
| **Routes** | `src/routes/` mirrors URL path. Dynamic segments use `$` prefix. |
| **Server functions** | `src/server/functions/` вЂ” `"use server"` functions called from client |
| **Shared UI** | `src/components/ui/` вЂ” shadcn primitives |
| **Business components** | `src/components/{feature}/` вЂ” feature-specific components |
| **Validation** | Zod schemas in `src/lib/validations.ts`, one per domain |
| **State** | `src/store/` вЂ” Zustand stores for client state |
| **DB** | Drizzle ORM in `src/server/db/` |
